# IBM HR Employee Attrition Analysis

## Overview
This project leverages People Analytics to understand why employees leave. By analyzing demographic trends and tenure, I provide actionable insights for HR departments to improve retention and workforce stability.

## Key Insights
- **Attrition Rate:** 16.12% of the workforce.
- **Average Tenure:** 7.01 years, indicating a generally stable workforce.
- **High-Risk Group:** Employees aged 28-35 show the highest density of attrition.
- **Departmental Focus:** Research & Development holds the majority of the workforce.
